## Brand Story

**Helix** represents the DNA double helix structure - the fundamental code of life. Just as DNA contains the blueprint for living organisms, Helix contains the blueprint for successful projects.

The name symbolizes:
- **Structure** - Organized, systematic approach
- **Evolution** - Projects that grow and adapt
- **Intelligence** - AI-powered insights
- **Uniqueness** - Every project has its own DNA
- **Connection** - Strands that bind everything together

---

## Visual Identity

### Logo Concept
A stylized DNA helix with:
- Gradient colors (purple to blue)
- Smooth, flowing curves
- Modern, minimalist design
- Can be animated (rotating helix)

### Color Palette

**Primary Colors**
- Purple: `#667eea` - Innovation, creativity
- Blue: `#4facfe` - Trust, intelligence
- Pink: `#f093fb` - Energy, passion
- Cyan: `#00f2fe` - Technology, future

**Secondary Colors**
- Aqua: `#a8edea` - Calm, clarity
- Yellow: `#ffc107` - Optimism, attention
- Green: `#4caf50` - Growth, success
- Red: `#ff5252` - Urgency, importance

**Background**
- Dark: `#0a0a1a` - Depth, sophistication
- Card: `rgba(30, 30, 50, 0.95)` - Glassmorphism

### Typography
- **Primary Font**: Inter
- **Weights**: 300 (Light), 400 (Regular), 600 (Semibold), 700 (Bold), 800 (Extrabold)
- **Headings**: Bold, gradient text
- **Body**: Regular, high contrast

## Usage Guidelines

### Do's ✅
- Use the full name "Helix" in formal contexts
- Include the tagline in marketing materials
- Use gradient effects on the name
- Emphasize the DNA/helix metaphor
- Highlight revolutionary features
- Show 3D visual effects

### Don'ts ❌
- Don't abbreviate to "HLX" or similar
- Don't use generic project management language
- Don't downplay the AI features
- Don't use flat, boring visuals
- Don't compare to traditional tools directly
- Don't forget the tagline
